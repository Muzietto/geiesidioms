# geiesidioms
Applicative functors (aka idioms) in crisp clear JavaScript and in [Khepri](http://khepri-lang.com).

Currently implemented:
- maybes
- lists
- curried unaries (aka (->)r)

Doubleclick on any html file and there you go...
